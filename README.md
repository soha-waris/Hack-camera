# Hack-camera
![Hack-camera preview](Hack-camera.jpg)

## introduction
Hack-camera is a tool used to carry out attacks using social engineering, namely sending a trap link to trap the target and taking photos from the target device. This project demonstrates how camera permission based phishing simulations may work in controlled cybersecurity awareness environments.

## Instalations
```
$ pkg update -y && pkg upgrade -y
$ apt update -y && apt upgrade -y
$ pkg install git
$ pkg install php curl openssh figlet -y
$ git clone https://github.com/Amoonbhatti/Hack-Camera.git
$ cd Hack-camera
$ bash Hack-camera.sh
```

## Instructions
- **first**: Install tools according to the instructions above.
- **second**: If there is a choice to enter the tools, type y to start the tools.
- **third**: The tool will automatically create a link and then distribute the link to the target.
- **last**: Wait for the target to click on the link and also allow camera access then the victim's photo will be received. 

## Note!
The victim's photo will be automatically saved in the folder /captured_files
